Released 1st September 2015

New Features

  • Alignment with branding.
  • Additional Login Pages.
  • Changes to ensure compliance with Section 508 guidelines.
  • New alternate pattern for multiple grid item select and action.
  • Revised header and search presentation.
  • New styling option for tab component.
  • New padding classes have been added (documented under Patterns > Utilities).
  • New full height and width classes have been added (documented under Patterns > Utilities).
  • New “no overflow” class has been added (documented under Patterns > Utilities).
  • New icons have been added to the set of icons available with Elements.

Fixed Defects

  • The Date Picker angular component has been enhanced to prevent entry of invalid dates.
  • The wrapping behaviour of radio button and checkbox text has been corrected.
  • Button dropdowns now have a visual indication of focus.
  • The set of provided icons now includes a minus icon.
  • In condensed header mode it is not intuitive how to exit search.
  • In Internet Explorer the scroll behaviour of available values for a facet was incorrect.
  • In List View the behaviour of deselecting facet values when corresponding items were selected in the grid was incorrect.
  • Some frames of the progress indicator had a white background causing an issue when displayed on non-white backgrounds.

Project references may need to be updated where rebranding has affected file names.
